
BackUpInit

BackUpInit — это консольная утилита на C++ для создания и управления бэкапами файловой системы.
Проект реализует собственную архитектуру команд, многопоточную обработку и механизм отслеживания изменений файлов через хеширование.

🚀 Возможности

- Инициализация бэкап-проекта
- Создание резервных копий файлов и директорий
- Отслеживание изменений через хеши файлов
- Проверка статуса бэкапа
- Многопоточная обработка (ThreadPool)
- Логирование операций

🧱 Архитектура

Проект построен по принципу Command Pattern:

- "ICommand" — базовый интерфейс команд
- "InitCommand" — инициализация проекта
- "RunCommand" — запуск бэкапа
- "StatusCommand" — проверка состояния

Дополнительно:

- "CommandDispatcher" — маршрутизация CLI-команд
- "BackupManager" — основная логика бэкапа
- "FileHash" — вычисление хешей файлов
- "ThreadPool" — многопоточность
- "Logger" — система логирования
- "TimeUtils" — работа со временем

🖥️ Использование

Программа работает через CLI:

Примеры команд:

backup init
backup run
backup status

(конкретные аргументы можно расширить в зависимости от реализации)

🧠 Как это работает

1. При инициализации создаётся конфигурация проекта
2. При запуске:
   - сканируются файлы
   - вычисляются хеши
   - сравниваются изменения
   - создаются копии изменённых файлов
3. Используется пул потоков для ускорения обработки
4. Все действия логируются
